Introduction
============

`django-allauth-thaid` is a package that adds the ThaiD provider to the `django-allauth` ecosystem.
ThaiD is the official digital identity system of the Department of Provincial Administration (DOPA) in Thailand.

Features
--------

- Seamless integration with `django-allauth`
- OAuth2 based authentication with ThaiD
- Extraction of Thai Personal ID (PID)
- Support for multiple identity scopes (name, birthdate, address, etc.)
