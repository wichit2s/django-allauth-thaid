============
Installation
============

You can install `django-allauth-thaid` via `pip` or `uv`.

Using Pip
---------

.. code-block:: bash

    pip install django-allauth-thaid

Using UV (Recommended)
----------------------

.. code-block:: bash

    uv add django-allauth-thaid

Dependencies
------------

- PHP >= 3.12
- Django >= 5.2
- django-allauth >= 0.63.0
- requests >= 2.32.5
